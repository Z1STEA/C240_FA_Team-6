About This Project

This website introduces the Republic Polytechnic FAQ Chatbot developed using Botpress. It explains how the chatbot works, what technologies were used, and how users can interact with it.

How the Chatbot Works

→
3. Knowledge Base is searched
→

How to Use the Chatbot

  1. Open the chatbot using the floating robot icon.
  2. Select a category or type your question naturally.
  3. The chatbot searches the knowledge base and returns the most relevant answer.

  • Where is the Library?
  • How do I pay my fees?
